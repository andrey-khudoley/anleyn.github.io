Подробная инструкция по установке OpenVPN-сервера для маскарадинга трафика

## Подготовка сервера

### Меняем пароль рута

```sh
passwd root
```

### Создаём пользователя

```sh
useradd anley
passwd anley
usermod -aG wheel anley
```
